# Diário de Desenvolvimento

Esta pasta reúne o histórico de desenvolvimento do **SmartMail-AI**.

Cada arquivo representa uma sessão de trabalho e registra a evolução do projeto ao longo do tempo.

## Objetivos do diário

* Documentar funcionalidades implementadas.
* Registrar decisões de arquitetura.
* Anotar problemas encontrados e suas soluções.
* Planejar os próximos passos do projeto.

O objetivo é manter um histórico claro da evolução do sistema, facilitando consultas futuras e demonstrando o processo de desenvolvimento do projeto.
